# Histroy
泛函分析是20世纪发展起来的一门学科，D. Hilbert,S. Banach,J. von Neumann等为此做出了主要贡献。  阅读书籍为《十遍函数与泛函分析基础》程其襄  
# Brief
泛函是函数的推广。函数是数到数之间的对应关系:  $x:->f(x)$
而泛函则是**函数和数之间的对应关系**，比如在 $[a,b]$上的连续函数全体记为$C[a,b]$,$C[a,b]$中的每一个函数都有一个积分值，即对任意的$f\in C[a,b]$,总有唯一的实数 $\int_{a}^{b}x^2 dx$与之对应，于是$C[a,b]$上的黎曼积分是一个泛函。  
同样考虑算子：函数空间和函数空间的对应关系。设$C^1 [a,b]$表示一阶连续可微函数所成的空间，那么微分就是一个从$C^1 [a,b]->C[a,b]$的算子
